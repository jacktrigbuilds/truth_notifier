This project was created to get updates on when President Trump has made a post on Truth Social. It will send a text message to your phone number when a post is made (polled every few minutes). The text will include the text content, links to articles/images, and a sentiment classification that attempts to capture stock market impact of the post The project is integrated with AWS EventBridge, Lambda, Comprehend, SQS and SNS.
